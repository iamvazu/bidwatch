Heading 1: 1. Executive Summary
None: BidWatch is a full-stack SaaS web application that automates the discovery, scoring, and analysis of California government janitorial contracts. It monitors 28+ procurement portals across the state, uses Claude AI to score bid fit, and delivers actionable proposal intelligence to small janitorial businesses competing for public sector contracts.
None: 
None: The problem it solves: a janitorial business owner today must manually check dozens of city, county, school district, transit, and state portals every day to find relevant contracts. Most miss bids because the process is manual, time-consuming, and fragmented. BidWatch consolidates this into one real-time dashboard with AI-powered go/no-go recommendations.

DB Note | Neon.tech (neon.tech) is a serverless PostgreSQL provider that works fully from India, has a free tier (0.5GB, 1 project), and connects via standard Postgres connection strings. No Supabase required. Alternative: Turso.tech for SQLite at the edge.

--- TABLE 5 ---
Component | Description
Scraper Service | Node.js workers on Railway/Render. Runs every 6hrs. Hits PlanetBids portals, Cal eProcure RSS, SAM.gov API. Parses HTML/JSON, extracts bid title, agency, deadline, document URLs, estimated value. Stores raw bids in PostgreSQL.
Deduplication | Each bid gets a hash of (agency + title + deadline). If hash exists in DB, skip. Prevents duplicate alerts.
AI Scoring Worker | After new bids are stored, a queue processor calls Claude API with bid metadata + user company profile to generate a fit score (0–100) and brief summary. Stores results in bids table.
Alert Engine | Checks DB every 30 minutes for: new bids (isNew=true, alerted=false), bids where deadline < 48hrs, high-score bids above user threshold. Sends email via Resend API. Marks alerted=true.
REST API | Express/Fastify server exposing endpoints for: GET /bids, GET /bids/:id/analysis, POST /bids/:id/draft, GET /portals, PUT /user/alerts, GET /pipeline
Next.js Frontend | BidWatch dashboard UI. Calls REST API. Deployed on Vercel. Includes: bid feed, AI analysis panel, pipeline kanban, alert settings, company profile.
PostgreSQL (Neon) | Stores: bids, users, company_profiles, alert_configs, pipeline_stages, ai_analyses, portal_registry
